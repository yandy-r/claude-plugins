# Frontend Patterns Command

Provide modern React and Next.js patterns for component design, state management, performance, and accessibility.

**Load and follow the `frontend-patterns` skill**, passing through `$ARGUMENTS`.

The skill covers a wide range of React/Next.js topics, including:

- Composition, compound components, render props, and custom hooks
- State management with `useState`, `useReducer`, and Context
- Performance: `React.memo`, `useMemo`, `useCallback`, `lazy`/`Suspense`, virtualization
- Forms with validation and error boundaries
- Animation with Framer Motion
- Accessibility: keyboard navigation and focus management
