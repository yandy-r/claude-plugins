# Python Patterns Command

Provide idiomatic Python patterns, PEP 8 conventions, and best practices for building robust, maintainable Python applications.

**Load and follow the `python-patterns` skill**, passing through `$ARGUMENTS`.

The skill covers a wide range of Python topics, including:

- EAFP vs LBYL and Pythonic control flow
- Type hints, `Protocol`, `TypeVar`, and `TypeAlias`
- `dataclasses`, `NamedTuple`, and `__slots__`
- Context managers and `with` blocks
- Decorators and `functools` utilities
- `asyncio`, threading, and multiprocessing trade-offs
- Idiomatic Python patterns and anti-patterns
