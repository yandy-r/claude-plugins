# Go Patterns Command

Provide idiomatic Go patterns, best practices, and conventions for building robust, efficient, and maintainable Go applications.

**Load and follow the `go-patterns` skill**, passing through `$ARGUMENTS`.

The skill covers a wide range of Go topics, including:

- Error handling and wrapping with `errors.Is` / `errors.As`
- Goroutines, channels, `context`, and `errgroup`
- Interface design ("accept interfaces, return structs")
- Functional options and struct embedding
- Package and module layout conventions
- Idiomatic Go patterns and anti-patterns
