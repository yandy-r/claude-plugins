# Frontend Slides Command

Create stunning, animation-rich, zero-dependency HTML presentations from scratch or by converting PowerPoint files.

**Load and follow the `frontend-slides` skill**, passing through `$ARGUMENTS`.

The skill covers:

- Viewport-fit slides (one slide = one viewport, no scrolling)
- Slide layout, animations, and keyboard/touch navigation
- Viewport fitting with `clamp()` and `100dvh`
- Distinctive design presets and themes
- Discovery-by-preview workflow
- Converting `.ppt` / `.pptx` to HTML with `python-pptx`
