# TypeScript Testing Command

Provide TypeScript testing patterns and TDD guidance using Vitest as the primary test runner.

**Load and follow the `ts-testing` skill**, passing through `$ARGUMENTS`.

The skill covers:

- Unit and integration tests with `describe` / `it` / `expect`
- Async tests with `vi.useFakeTimers`
- Parameterized tests via `test.each`
- Property-based testing with `fast-check`
- Mocking with `vi.mock`, `vi.fn`, and `vi.spyOn`
- Type-level testing with `expectTypeOf` and `tsd`
- Benchmarks with `vitest bench` and v8 coverage
