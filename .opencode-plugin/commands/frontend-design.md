# Frontend Design Command

Create distinctive, production-grade frontend interfaces with intentional visual direction.

**Load and follow the `frontend-design` skill**, passing through `$ARGUMENTS`.

The skill covers:

- Typography pairings and hierarchy
- Color systems and contrast
- Spacing rhythm and layout composition
- Motion choreography and atmosphere
- Design directions: brutalist, editorial, luxury, playful, maximalist
- Upgrading bland or generic UIs into memorable interfaces
