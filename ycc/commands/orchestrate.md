# Orchestrate Command

Decompose and orchestrate a complex task across multiple specialized agents.

**Load and follow the `ycc:orchestrate` skill**, passing through `$ARGUMENTS`.

The skill analyzes the task, breaks it into independent subtasks, deploys an agent team in parallel batches with inter-agent communication, and synthesizes the results.
