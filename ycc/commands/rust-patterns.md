# Rust Patterns Command

Provide idiomatic Rust patterns, ownership, error handling, traits, concurrency, and best practices for building safe, performant applications.

**Load and follow the `ycc:rust-patterns` skill**, passing through `$ARGUMENTS`.

The skill covers a wide range of Rust topics, including:

- Ownership, borrowing, and lifetimes
- Error handling with `Result`, `thiserror`, and `anyhow`
- Traits, generics, and trait objects
- Concurrency with `Arc`, `Mutex`, channels, and async/await
- Idiomatic Rust patterns and anti-patterns
