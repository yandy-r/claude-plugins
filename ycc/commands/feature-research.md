# Feature Research Command

Research the specified feature and produce a `feature-spec.md`.

**Load and follow the `ycc:feature-research` skill**, passing through `$ARGUMENTS`.

The skill deploys a coordinated research team to gather requirements, external API context, and prior art, then synthesizes the findings into a structured feature specification.
