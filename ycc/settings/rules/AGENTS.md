# Agent Rules (User-Global)

The canonical rules live in [`CLAUDE.md`](CLAUDE.md) (sibling file).

This file exists so agent-runtime tools that default to `AGENTS.md` (Cursor, Codex,
opencode, Windsurf, etc.) pick up the same rules without duplicating prose.

## Agent-runtime notes

- **Claude Code** reads `CLAUDE.md` from its user-global config dir
  (`~/.claude/CLAUDE.md`).
- **Cursor** reads user-global `AGENTS.md` (`~/.cursor/AGENTS.md`) and project-local
  `.cursor/rules/**/*.mdc`.
- **Codex** reads `~/.codex/AGENTS.md`.
- **opencode** reads `~/.config/opencode/AGENTS.md`.
- When an MCP server fits the task, prefer it; read each tool's schema first.

## Sync policy

`CLAUDE.md` is the source of truth. Do not edit this file except to add
agent-runtime-specific notes. Never duplicate `CLAUDE.md` content here.
