# Release 2.5.0 — 2026-04-19

## Summary

Adds `/ycc:git-cleanup` — a dry-run-first auditor and cleaner for stale git
resources (branches, worktrees, remote-tracking refs, stashes, tags, PRs,
issues) across GitHub and GitLab that cross-references active code before
proposing deletions. Tightens the release surface with the `--hooks` flag for
the Claude install target, a `--worktree` flag for task isolation, and an
enforced skill↔command pairing policy in the validator.
