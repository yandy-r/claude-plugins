# Release 3.1.0 ‚Äî 2026-04-23

## Summary

Adds an interactive `/ycc:quick-review` ‚Üí `/ycc:quick-fix` loop and a `--quick`
flag on `/ycc:code-review` for fast, artifact-free reviews. Expands the
`/ycc:formatters` surface (aliases, auto-sync hooks, pinned shellcheck) and
hardens Codex bundle generation so team rewrite semantics survive round-trips.

## Added

- `feat(quick-review)`: interactive inline review with confirm flow ‚Äî new
  `/ycc:quick-review` skill and command that reviews uncommitted changes,
  prints findings inline, and prompts before writing anything
  (`a048e60`, #77).
- `feat(quick-review)`: direct quick-fix handoff ‚Äî `/ycc:quick-review` can
  hand its inline findings straight to `/ycc:quick-fix` without producing a
  review artifact (`0135dd2`).
- `feat(code-review)`: `--quick` flag for on-the-fly reviews ‚Äî skips the
  artifact write and posts findings inline (`18570e8`).
- `feat(ycc)`: expanded `/ycc:formatters` alias surface and auto-sync hooks
  for package.json / Makefile / justfile targets (`1246642`).
- `docs(ycc)`: `/ycc:init --formatters` flag documented on the init command
  (`8dacc1f`).

## Changed

- Codex bundle regenerated after slash-command additions and the Codex team
  rewrite fix (`c8ddf6f`, `58187d1`, `0955a75`, `1296f4e`).
- Cursor / Codex / opencode bundles resynced after prettier normalization in
  `/ycc:code-review` (`63fbf6d`).
- Formatter stack docs regenerated after the formatters alias expansion
  (`8838e2b`).
- Settings and helper scripts updated to support the new hooks
  (`423f120`, `8fb0f32`).

## Removed

None.

## Fixed

- `fix(codex)`: preserve team rewrite semantics when generating Codex
  agents ‚Äî prevents round-trip loss of multi-agent team directives
  (`c87ed91`).
- `fix(formatters)`: split biome and prettier ownership so each tool owns a
  non-overlapping file set (`2a8bc27`).
- `fix(formatters)`: unify pinned shellcheck setup across `scripts/style.sh`
  and CI (`86a8484`).
- `fix(ycc)`: allow model invocation of the `init` and `formatters` skills
  (previously blocked by description triggers) (`2424b24`).
